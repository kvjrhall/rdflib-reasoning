# OWL 2 Mapping to RDF

- This directory covers the bidirectional mapping between OWL 2 structural constructs and RDF graph form.
- `INDEX.md` is the primary Development Agent-facing artifact. Use it to locate mapping constructs and section spans quickly.
- `index-data.json` is the machine-readable sidecar for construct-level joins and other tooling.
- `create-index.py` regenerates `INDEX.md` and `index-data.json`.
- Use this spec when reasoning about structural-element serialization, RDF parsing, and construct-level mapping anchors.
