# OWL 2 RDF-Based Semantics

- This directory is the primary source for normative RDF-based semantic-condition sections and item-level semantic condition anchors.
- `INDEX.md` is the primary Development Agent-facing artifact. Use it to navigate semantic-condition sections, semantic-condition items, axiomatic-triples sections, and correspondence-theorem areas.
- `index-data.json` is the machine-readable sidecar for section/item joins and crosswalk generation.
- `create-index.py` regenerates `INDEX.md` and `index-data.json`.
- Use this spec when you need the semantic foundation behind OWL 2 RL rules or axiomatic-triple relationships.
