# rdflib-reasoning

`rdflib-reasoning` is a monorepo for Python packages and notebooks used to study how
tool-grounded Research Agents interact with RDF graphs and formal reasoning systems.

This documentation set gives a fast path into the repository:

- Start with [Getting Started](getting-started.md) for installation and common commands.
- Use [Package Guides](package-guides/index.md) for subsystem-level orientation.
- Use [API Reference](api/index.md) for generated Python documentation from docstrings and
  type hints.
- Use the [Development Docs](development/architecture.md) pages for intended architecture and
  planned scope.
