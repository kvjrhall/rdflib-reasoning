# Axioms Package

The `rdflib-reasoning-axioms` package defines the graph-backed Pydantic model hierarchy used as
the schema-facing contract between middleware and Research Agents.

Use this package when you need:

- immutable graph-scoped data models
- typed structural elements that can emit RDF triples and quads
- a JSON-schema-friendly boundary for runtime tools and middleware state

For generated API docs, see {doc}`../autoapi/axiom/index`.
