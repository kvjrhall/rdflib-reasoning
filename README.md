# Benchmarking Tool-Grounded Agents on Formal Reasoning Tasks with External Knowledge Retrieval

> When do tool-grounded agent harnesses outperform direct prompting on multi-step formal reasoning tasks that require external knowledge retrieval, knowledge-base updates, and verifiable inference?

One cannot easily axiomatize RDFLib graphs, nor can they easily implement a reasoner based on them.
This repository intends to fix that.
